Introducción
============

    Éste paquete tiene la finalidad de ilustrar como se utiliza buildout
    en la construcción de un sitio con Plone 4.

    Se basa en el buildout de desarrollo para el paquete collective.nitf
    y al cual hemos agregado un par de paquetes que desarrollaremos en
    el fin de realizar un tutorial de programación con Plone 4 y Diazo
    (mediante plone.app.theming).

    Los paquetes incluidos en el buildout de desarrollo son:

    collective.nitf .-
                Los archivos se encuentran alojados en la carpeta "src"
    ejercicios.plone4 .-
                Se descarga de http://github.com/saibatizoku/ejercicios.plone4
    ejercicios.plone4.tema .-
                Se descarga de http://github.com/saibatizoku/ejercicios.plone4.tema
